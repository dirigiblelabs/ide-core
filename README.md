# Core Services for Eclipse Dirigible WebIDE

Core Services for the [Eclipse Dirigible](https://github.com/eclipse/dirigible) WebIDE v3.x and above.

Build the [Eclipse Dirigible](https://github.com/eclipse/dirigible), deploy on e.g. Tomcat Web Container and go to location:

> http://localhost:8080/services/v3/web/ide/index.html
